# Arce_Tools
Arce test software series repository, licensed under MIT. See License for details.  
If GPGPU tests cannot run, DirectX End-User Runtime needs to be installed. For details, see the official documentation from Microsoft, NVIDIA, AMD, Intel, or the device manufacturer.  
All C++ software in this repository uses C++17 or later, and all Python software (if any) uses Python 3.13.9 or later.
